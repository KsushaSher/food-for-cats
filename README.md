# Cat Energy — Landing Page

A responsive landing page for a fictional functional cat food brand **Cat Energy**.  
Built from a Figma design without CSS/JS frameworks.

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Semantic HTML markup
- Custom JavaScript "Before / After" image comparison slider
- Yandex Map with custom marker
- Pixel-perfect approximation to the design
- Clean SCSS → compiled into CSS
- Cross-browser tested (Chrome, Firefox, Safari, Edge)
